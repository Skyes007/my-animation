📋 DROP/TRUNCATE/DELETE 语句标准格式
R DROP TABLE [IF EXISTS] table_name; # 删除整个表，包括表结构和数据
R TRUNCATE TABLE table_name; # 清空表数据，保留表结构
R DELETE FROM table_name [WHERE condition] ; # 删除表数据，可指定条件
对比 TRUNCATE TABLE 和 DELETE FROM 命令的区别：
TRUNCATE TABLE 不记录事务操作，执行速度快，会重置自增字段；
DELETE FROM 可以结合 WHERE 子句删除部分数据，会记录事务操作，执行速度相对较慢。
