步骤1: 创建数据库

首先，我们需要创建一个名为STUINFO的数据库，用于存储学生信息相关的数据。

SQL 编辑器 MySQL
提示：使用CREATE DATABASE语句创建数据库
示例：CREATE DATABASE stuinfo;
✓ 正确！SQL语句执行成功。
✗ 错误！请检查SQL语句是否正确。
CREATE DATABASE 标准语句格式
CREATE DATABASE database_name
[CHARACTER SET charset_name]
[COLLATE collation_name];
参数说明：
  • CREATE DATABASE - 创建数据库的关键字
  • database_name - 数据库名称（此处为stuinfo）
  • CHARACTER SET - 可选，指定字符集
  • COLLATE - 可选，指定排序规则
