步骤1: 完整插入

完整插入语句需要指定所有列的值，包括自增列。注意：自增列的值如果与已有值冲突，插入会失败。

SQL 编辑器 MySQL
提示：使用INSERT INTO语句插入数据，注意自增列的值不能重复
示例：INSERT INTO students(student_id, student_name, sex, student_age, birthday) VALUES(1, '张三', '男', 19, '2002-12-1');
✓ 正确！SQL语句执行成功。
✗ 错误！请检查SQL语句是否正确。
INSERT INTO 标准语句格式
INSERT INTO table_name
(column1, column2, ...)
VALUES
(value1, value2, ...);
参数说明：
  • INSERT INTO - 插入数据的关键字
  • table_name - 表名称（此处为students）
  • column1, column2, ... - 要插入数据的列名
  • VALUES - 指定要插入的值
  • value1, value2, ... - 对应列的值
students表结构
列名 数据类型 约束
student_id INT PRIMARY KEY, AUTO_INCREMENT
student_name VARCHAR(50) NOT NULL
sex CHAR(2) NOT NULL
student_age INT DEFAULT 18
birthday DATE NULL
