步骤1: 基本用法

使用CREATE TABLE...SELECT语句创建新表并复制所有数据，新表结构与SELECT查询结果相同。

SQL 编辑器 MySQL
提示：使用CREATE TABLE new_table AS SELECT * FROM source_table;
示例：CREATE TABLE T_Student_Backup AS SELECT * FROM T_Student;
✓ 正确！SQL语句执行成功。
✗ 错误！请检查SQL语句是否正确。
CREATE TABLE...SELECT 标准语句格式
CREATE TABLE table_name
[(column_definition, ...)]
AS
SELECT column1, column2, ...
FROM table_name
[WHERE condition]
[GROUP BY column]
[ORDER BY column [ASC|DESC]]
[LIMIT number];
参数说明：
  • CREATE TABLE - 创建表的关键字
  • table_name - 新表的名称
  • [(column_definition, ...)] - 可选的列定义
  • AS - 指定数据来源
  • SELECT - 选择数据的关键字
  • column1, column2, ... - 要查询的列名，使用*表示所有列
  • FROM - 指定数据来源表
  • WHERE - 筛选条件（可选）
  • GROUP BY - 分组（可选）
  • ORDER BY - 排序（可选）
  • LIMIT - 限制结果数量（可选）
