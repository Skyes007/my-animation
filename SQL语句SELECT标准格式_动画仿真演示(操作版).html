步骤1: 基本查询

使用SELECT * 查询所有列，FROM指定数据来源表。这是最简单的SELECT语句形式。

SQL 编辑器 MySQL
提示：使用SELECT * FROM table_name查询所有列
示例：SELECT * FROM students;
✓ 正确！SQL语句执行成功。
✗ 错误！请检查SQL语句是否正确。
SELECT 标准语句格式
SELECT column1, column2, ...
FROM table_name
[WHERE condition]
[GROUP BY column]
[HAVING condition]
[ORDER BY column [ASC|DESC]]
[LIMIT number];
参数说明：
  • SELECT - 选择数据的关键字
  • column1, column2, ... - 要查询的列名，使用*表示所有列
  • FROM - 指定数据来源表
  • WHERE - 筛选条件（可选）
  • GROUP BY - 分组（可选）
  • HAVING - 分组后筛选（可选）
  • ORDER BY - 排序（可选）
  • LIMIT - 限制结果数量（可选）
