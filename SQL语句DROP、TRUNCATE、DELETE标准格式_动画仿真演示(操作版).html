步骤1: DELETE语句 - 条件删除

使用DELETE FROM语句删除满足特定条件的记录。DELETE语句可以指定WHERE子句来筛选要删除的记录。

SQL 编辑器 MySQL
提示：使用DELETE FROM table_name WHERE condition删除满足条件的记录
示例：DELETE FROM students WHERE student_age > 19;
✓ 正确！SQL语句执行成功。
✗ 错误！请检查SQL语句是否正确。
DROP/TRUNCATE/DELETE 标准语句格式
DELETE FROM table_name
[WHERE condition];

TRUNCATE TABLE table_name;

DROP TABLE [IF EXISTS] table_name;
参数说明：
  • DELETE FROM - 删除表中的数据
  • table_name - 要操作的表名
  • WHERE - 筛选条件（可选）
  • TRUNCATE TABLE - 清空表数据，保留表结构
  • DROP TABLE - 删除整个表，包括表结构和数据
  • IF EXISTS - 如果表存在才删除（可选）
students表结构
列名 数据类型 约束
student_id INT PRIMARY KEY AUTO_INCREMENT
student_name VARCHAR(50) NOT NULL
sex CHAR(2) NOT NULL
