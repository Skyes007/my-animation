SELECT 集函数与分组查询格式
SELECT column1, column2, ...
FROM table_name
[WHERE condition]
[GROUP BY column]
[HAVING condition]
[ORDER BY column [ASC|DESC]]
[LIMIT number];
常用集函数:
  • COUNT() - 统计记录数或非NULL值数量
  • SUM() - 计算数值列的总和
  • AVG() - 计算数值列的平均值
  • MAX() - 返回数值列的最大值
  • MIN() - 返回数值列的最小值
