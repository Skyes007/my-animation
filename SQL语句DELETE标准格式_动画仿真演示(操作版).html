步骤1: 按主键删除

按主键删除是最精确的删除方式，可以确保只删除指定的记录。使用WHERE条件指定要删除的特定记录。

SQL 编辑器 MySQL
提示：使用DELETE FROM ... WHERE ... 格式删除特定记录
示例：DELETE FROM students WHERE student_id = 1;
✓ 正确！SQL语句执行成功。
✗ 错误！请检查SQL语句是否正确。
DELETE 标准语句格式
DELETE FROM table_name
WHERE condition;
参数说明：
  • DELETE FROM - 删除数据的关键字
  • table_name - 表名称（此处为students）
  • WHERE - 指定删除条件（可选，但强烈建议使用）
  • condition - 删除条件，如 student_id = 1
注意： 不带WHERE条件的DELETE语句将删除表中的所有数据，请谨慎使用！

列名 数据类型 约束

student_name VARCHAR(50) NOT NULL
