步骤1: 创建基本表结构

首先，我们需要创建T_Student表的基本结构，包含所有字段但不包含约束。

SQL 编辑器 MySQL
提示：使用CREATE TABLE语句创建表结构
示例：CREATE TABLE T_Student (S_number CHAR(10), S_name CHAR(20), Sex CHAR(2), Birthday DATETIME, BirthPlace VARCHAR(50), Ploitics VARCHAR(50), Class_number INT, MobilePhone VARCHAR(50), Dormitory VARCHAR(50));
✓ 正确！SQL语句执行成功。
✗ 错误！请检查SQL语句是否正确。
CREATE TABLE 标准语句格式
CREATE TABLE table_name (
  column1 datatype [constraints],
  column2 datatype [constraints],
  ...
  [PRIMARY KEY (column)]
  [FOREIGN KEY (column) REFERENCES table(column)]
  [CHECK (condition)]
)[ENGINE = engine_name];# 存储引擎（可选）默认ENGINE=InnoDB;
常用约束：
  • PRIMARY KEY - 主键约束
  • NOT NULL - 非空约束
  • UNIQUE - 唯一约束
  • CHECK - 检查约束
  • DEFAULT - 默认值
  • FOREIGN KEY - 外键约束
外键级联操作：
  • ON DELETE CASCADE - 主表删除时同步删除子表记录
  • ON UPDATE CASCADE - 主表更新时同步更新子表记录

级联操作说明
ON DELETE CASCADE: 当T_Class表中的班级记录被删除时，T_Student表中对应的学生记录也会自动删除。
ON UPDATE CASCADE: 当T_Class表中的班级编号更新时，T_Student表中对应的班级编号也会自动更新。
示例：
删除班级1 → 自动删除班级1的所有学生
更新班级1为班级10 → 自动更新学生班级编号为10
